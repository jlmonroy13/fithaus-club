This file defines the source-of-truth artifacts for the AI product pipeline.

## Source of Truth
- Discovery
- System Design
- Screen Specs
- Refined Stories
- Planning
- Planning Export
- AI Task Briefs (per ticket)

## Delivery Governance

- Stage 7 output is the source for GitHub milestones and tickets.

## Forbidden

- Changing flows without explicit approval
- Ignoring established technical constraints
